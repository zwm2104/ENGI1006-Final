We have had 5 homeworks in this course (8 assignments, if you count each part separately).

HW1 covered introductory theory and Python syntax. In A, we covered binary numbers, algorithms, and logic. In B, we covered lists.

HW2 covered sorting algorithms, simulations, and CSVs. In A, we covered time complexity associated with various sorting algorithms. In B, we ran a physics simulator and wrote functions to read in information about an input CSV file.

HW3 covered recursion. A involved exercises in time complexity. B covered recursive functions via a percolation simulator (and some other functions to process CSV files).

HW4 covered object oriented learning via a course simulator.

HW5 pulled from previous homeworks and used various data analysis libraries to create a machine-learning programs.

My favorite assignment was the midterm, actually. It was really fun to see the payoff of the functions we wrote represented visually. Plus, I felt like it gave me insight into how the apps that I know and love may have similar underlying code!
